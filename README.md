# VerifMotDePasse
Ce projet est un vérificateur de mot de passe à double authentification en Python. L’utilisateur a un nombre de tentatives défini pour entrer le mot de passe. En cas d’échec, un code de secours de trois chiffres est généré et sauvegardé dans code_secret.txt. L’utilisateur doit saisir ce code pour valider l’accès.
